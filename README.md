## How to Deploy on Render (24hr Ranking Goal)

### Step 1: Create GitHub Repository
1. Go to github.com and create a new repository
2. Upload all files from this folder
3. Commit and push

### Step 2: Deploy on Render
1. Go to render.com and sign up/login
2. Click "New" → "Static Site"
3. Connect your GitHub repository
4. Set build command: (leave empty for static)
5. Set publish directory: `/` (root)
6. Click "Create Static Site"

### Step 3: Add Custom Domain (Optional but Recommended)
1. Buy a domain (e.g., jayamurugan.dev) from Namecheap/GoDaddy
2. In Render dashboard, go to your site settings
3. Add custom domain and follow DNS instructions
4. Update canonical URL in index.html

### Step 4: Google Search Console (CRITICAL for 24hr indexing)
1. Go to search.google.com/search-console
2. Add your property (domain or URL prefix)
3. Verify ownership (HTML file upload or DNS record)
4. Submit sitemap: `https://yoursite.com/sitemap.xml`
5. Request indexing for your homepage URL

### Step 5: Speed Up Indexing
- Share your site on Twitter/X, LinkedIn, Reddit
- Create a GitHub profile README linking to your site
- Submit to Google via URL Inspection Tool in Search Console
- Ping sitemap: `https://www.google.com/ping?sitemap=https://yoursite.com/sitemap.xml`
